# Mein Steckbrief

## Los geht's
1. VS Code öffnen
2. Rechtsklick auf `index.html` → **Open with Live Server**
3. Immer speichern mit `Ctrl+S` – die Seite aktualisiert sich automatisch!

---

## Aufgaben

### index.html – Inhalte ausfüllen
- [ ] Deinen Namen in `<h1>` schreiben
- [ ] Kurze Beschreibung anpassen
- [ ] Alter, Wohnort, Schule, Lieblingsfarbe eintragen
- [ ] Deine 3 Hobbys eintragen
- [ ] Ein Bild einfügen (Anleitung steht im Code)
- [ ] Warum Informatik? – 2–3 Sätze schreiben
- [ ] Lieblingslink ändern
- [ ] Namen im Footer eintragen

### style.css – Aussehen gestalten
- [ ] Hintergrundfarbe der Seite ändern
- [ ] Header-Farbe ändern
- [ ] Schriftfarbe ändern
- [ ] Bild-Breite und runde Ecken anpassen
- [ ] Eigene Ideen ausprobieren!
